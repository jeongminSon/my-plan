## 상세 설명
my-plan은 "지금 무엇을 할지"에 집중하도록 설계된 미니멀 할일 앱입니다.

- **빠른 추가**: 앱을 열면 바로 입력 — 제목만으로 3초 안에 등록
- **오늘 보기**: 오늘 할 일이 첫 화면에. 마감 임박 순 정렬
- **진행 통계**: 오늘 완료율과 이번 주 완료 수로 동기부여
- **우선순위·색상**과 **하위 할일(체크리스트)** 로 큰 일도 잘게 쪼개기
- **반복 할일**: 매일/매주/매월 — 완료하면 다음 일정 자동 생성
- **알림**: 원하는 시각에 로컬 리마인더 1회
- **목록/프로젝트**로 분류, **다크 모드** 지원
- **프라이버시 우선**: 기본 로컬 저장, 분석 SDK 없음, 푸시 토큰 미발급
